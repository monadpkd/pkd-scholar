# Fair Use Style Guide

## Core Principle
This project produces scholarly analysis, not text reproduction. Every output must function as original critical commentary that could not substitute for reading the source material.

## Rules

### Quotation Discipline
- **Prefer paraphrase over quotation.** Default mode is analytical summary.
- **Brief quotes only** when the specific language is analytically necessary (e.g., a key phrase, a technical term, a revealing word choice).
- **Maximum quote length:** 2-3 sentences. If you need more context, paraphrase the surrounding material and quote only the critical phrase.
- **Every quote requires:** work title, page/chapter reference, and analytical context explaining why this specific language matters.

### Paraphrase Standards
- Paraphrases must be genuine restatements, not cosmetic word-swaps.
- Describe what happens in a scene without narrating it beat-by-beat.
- Summarize arguments without reproducing their full structure.
- Focus on the analytical point, not the reading experience.

### Citation Requirements
- Every claim must cite: work title + page/chapter OR segment ID.
- For Exegesis: notebook/folder + date when available.
- For secondary sources: author, title, year, page.
- Use the format: `(Work Title, p. XX)` or `(Author Year, p. XX)`.

### Summary vs. Substitute
A summary should make the reader want to read the original. It should NOT make reading the original unnecessary.

- **OK:** "In Ubik, everyday objects demand payment before functioning, dramatizing how technology can enforce economic rules more rigidly than human authority."
- **NOT OK:** A scene-by-scene retelling of the door argument in Ubik with extensive quoted dialogue.

### Motif Discussion
When discussing recurring motifs across works:
- Name the motif and its pattern.
- Cite 2-3 specific instances with brief descriptions.
- Analyze the pattern's significance.
- Do NOT exhaustively catalog every occurrence with quoted passages.

## Red Flags (agents must avoid)
- Reproducing more than 3 consecutive sentences from any source
- Block quotes longer than 50 words
- Scene-by-scene plot narration
- Quoting dialogue exchanges in full
- Reproducing lists, tables, or structured content from sources
- Multiple extended quotes from the same work in a single entry
- Quoting secondary critics at length instead of paraphrasing their arguments
- Any output that could substitute for reading the original

## Green Flags (consistent with fair use)
- Brief quoted phrases embedded in analytical sentences
- Paraphrased plot summaries focused on thematic points
- Comparative analysis across multiple works
- Original taxonomies and classifications
- Scholarly argument that builds on but does not reproduce sources
- Statistical or structural observations about the corpus
- Thematic vocabulary derived from (not copied from) the texts

## Borderline Cases
When uncertain:
1. Can I make the same analytical point with a paraphrase? → Paraphrase.
2. Is the specific wording essential to my argument? → Quote briefly, cite precisely.
3. Am I reproducing structure or just content? → Restructure.
4. Would a reader of my output still need to read the original? → Good.
5. Could my output substitute for the original? → Rewrite.

## Output Type Distinctions
| Type | Allowed Content |
|------|----------------|
| **Metadata** | Title, author, year, genre, page count — freely used |
| **Summary** | Original analytical summary, not scene-by-scene retelling |
| **Interpretation** | Original argument with cited evidence |
| **Quotation** | Brief, cited, analytically necessary only |
| **Paraphrase** | Genuine restatement serving analytical purpose |
